### 2. Django Tutorial

- **Description:** A tutorial provided as part of the CS50W course focusing on building a basic Django application.
- **Link:** [Django Tutorial](https://github.com/Abeehimr/CS50W_Projects/tree/main/django_tutorial_cs50)
